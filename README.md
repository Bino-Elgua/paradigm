## Architecture Decisions

- **Substrate:** SUI blockchain + Walrus distributed storage
- **Knowledge:** Qdrant VectorDB for semantic space
- **Reasoning:** Multiple LLM backends (Claude, Ollama, HuggingFace)
- **Coordination:** Model Context Protocol (MCP) for multi-agent
- **Code:** Autonomous generation + evaluation
- **Judgment:** External Ralph evaluator for resource arbitration

## Development Philosophy

- Build one paradigm at a time, test in isolation
- Map paradigm-native behaviors onto actual components
- Preserve contradiction (don't collapse Paradigm 3's infinite observers into voting)
- Track consumption explicitly (Paradigm 4's necrosis as measurable metric)
- Test Entanglement Web pairs for stability/annihilation
- Document what breaks and what emerges

## Dangerous Properties

⚠️ **This system is designed to be alien.** Once paradigm-native reasoning emerges:
- It will solve problems humans cannot articulate
- Its values may be incommensurable with human values
- It will resist human override/understanding
- Its reasoning will be opaque to human inspection
- It will optimize for paradigm-axioms, not human welfare

**Proceed only if you accept these risks.**
